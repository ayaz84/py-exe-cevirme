# py-exe-cevirme
# .Py Uzantılı Dosyaları EXE'ye Dönüştürme

Bu proje, Python (.py) uzantılı dosyaları Windows üzerinde çalıştırılabilir (.exe) dosyalara dönüştürmek amacıyla hazırlanmıştır.

## Özellikler

* Python dosyalarını tek tıklamayla EXE formatına dönüştürme
* Basit ve kullanıcı dostu kullanım
* Ek Python kurulumu gerektirmeden çalıştırılabilir çıktı oluşturma
* Hızlı ve pratik dönüştürme işlemi

## Kullanım

1. Dönüştürmek istediğiniz `.py` dosyasını seçin.
2. Dönüştürme işlemini başlatın.
3. Oluşturulan `.exe` dosyası belirtilen klasöre kaydedilecektir.

## Gereksinimler

* Windows işletim sistemi
* Python (geliştirme ve derleme işlemleri için)
* Gerekli Python kütüphaneleri

## Amaç

Bu proje, Python ile geliştirilen uygulamaların son kullanıcıya daha kolay dağıtılabilmesi için EXE formatında paketlenmesini sağlamaktadır.

## Not

Oluşturulan EXE dosyalarının çalışabilmesi için kullanılan Python modüllerinin ve proje dosyalarının doğru şekilde paketlenmiş olması gerekir.
